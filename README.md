## Introdução

Este repositório contém exemplos didáticos em C# sobre criptografia, hash e conversões. Incluindo uma ferramenta experimental para testar diferentes tipos de criptografia.

## Projetos

### ConsoleTestConvertions
- **Descrição**: Programa didático sobre conversão para base 64.
- **Como usar**: Abra o projeto ConsoleTestConvertions.sln no Visual Studio.

### ConsoleTestHashes
- **Descrição**: Programa didático sobre uso de hashes padrões.
- **Como usar**: Abra o projeto ConsoleTestHashes.sln no Visual Studio.

### CryptoAES
- **Descrição**: Programa console desenvolvido para cifrar/decifrar arquivos utilizando criptografia AES.
- **Como usar**: Utilize o Visual Studio, abra o .sln do projeto Cryptool.

### CryptoDES
- **Descrição**: Programa console desenvolvido para cifrar/decifrar arquivos utilizando criptografia DES.
- **Como usar**: Utilize o Visual Studio, abra o .sln do projeto Cryptool.

### Cryptool
- **Descrição**: Interface para testes dos programas de criptografia CryptoAES e CryptoDES.
- **Como usar**: Utilize o Visual Studio, abra o .sln do projeto Cryptool.
